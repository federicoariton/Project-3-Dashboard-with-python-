
The Dash app can be deployed by running the script using Python

Open the Anaconda Prompt and install:

  pip install dash dash-bootstrap-components plotly pandas openpyxl

And after Run the script
  
 python Dahboard.py


Access the Dashboard:

Open the URL http://127.0.0.1:8063/ in a web browser to interact with the dashboard.
i use the port 8063,but you can change it
